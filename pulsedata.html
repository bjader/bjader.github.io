I was first introduced to Java in my 3rd year of university, where I took a course entitled: "Scientific Computing using Object Oriented Languages".

For the final exam I was tasked with analysing data from a set of 100 "pulses", each one representing a group of particles travelling down a particle accelerator. This involved reading such data from a URL, parsing it into appropriate data structures and applying various analysis methods as desired.

Given 3 hours, my solution to this problem can be seen here. In order to make my program both general and modular, I created a custom PulseData object to store the inputted data, as well as various analysis classes. The main method to run the whole program is then contained in the FinalExam.java file.
